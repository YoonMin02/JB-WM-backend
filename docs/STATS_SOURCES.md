# STATS_SOURCES · 통계/기준 데이터 소스 (분류 ②)

에이전트의 **근거 있는 판단**(환각 방지·설명가능성)을 위한 한국 공개 통계 소스 목록입니다.
데이터 분류 ②(통계/기준)에 해당하며, RAG가 아니라 **파라미터 쿼리 도구**(`get_population_stat`)로 노출합니다. ([02_SYSTEM_ARCHITECTURE](02_SYSTEM_ARCHITECTURE.md), [06_TOOL_CONTRACTS](06_TOOL_CONTRACTS.md) 참고)

> ⚠️ 사용 전 각 출처의 **라이선스·이용약관**을 반드시 확인하세요. 대다수 공공데이터는 공공누리(KOGL) 유형이며 출처 표시가 요구됩니다. 보험개발원·일부 의료 데이터는 별도 약관이 있을 수 있습니다.

## B. 다운로드 데이터셋 (파일로 적재)

실시간 OpenAPI가 제한적이라, 필요한 조각을 받아 `PopulationStat` 테이블/파일로 시드합니다.

| 소스 | 운영 | 내용 | 형태 |
|---|---|---|---|
| **KNHANES 국민건강영양조사** | 질병관리청 | 연령대별 혈압·BMI·만성질환 유병률 등 건강지표 | 원시자료/통계표 다운로드 |
| **보험개발원 KIDI** | 보험개발원 | 경험생명표, 위험률, 보험통계 | 공개 통계표/보고서 |
| **건강검진통계** | 국민건강보험공단 | 연령대별 검진 결과 분포 | 통계표 / 공공데이터포털 경유 |

## MVP에서 실제로 쓸 조각 (권장)

전체 연동은 과하므로, 시나리오에 필요한 **몇 지표만** 뽑아 시드합니다.

| 지표 | 출처 | 용도 (시나리오) |
|---|---|---|
| 65–69세 평균 순자산·부채 | KOSIS 가계금융복지조사 | "또래 대비 현금흐름 완충 부족" 근거 |
| 연령대별 고혈압/심혈관 유병률 | KNHANES | 건강 이벤트의 통계적 맥락 |
| 동일 질병 평균 의료비 | HIRA(심평원) | 의료비 대비 자금 플랜의 근거 |
| 경험생명표(기대여명) | KIDI | 장기 생애설계(LifePlan) 가정 |

### 적재 → 노출 흐름

```
공개 데이터(다운로드/API)  →  PopulationStat 테이블/파일(시드)  →  get_population_stat(age_band, metric)  →  에이전트 read-only 조회
```

- `get_population_stat`은 **정형 쿼리**입니다(정확한 값 반환). 통계 전체를 프롬프트에 넣지 않습니다.
- 반환 시 **출처(source)·기준시점(as_of)을 함께** 반환하여 설명가능성·신뢰성을 확보합니다.
- 실시간 API 호출은 슬라이스 5+ 옵션. MVP는 시드로 충분.

## 의료 통계 사용 시 경계 (중요)

의료비·유병률 통계는 **참고 정보**로만 제공하며, **의료 권고를 생성하지 않습니다** ([10_SECURITY_PRIVACY](10_SECURITY_PRIVACY.md), 의료 경계 원칙). 항상 출처를 명시하고, 의료 판단 주체는 고객·주치의임을 분명히 합니다.
